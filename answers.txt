                                     Authentication
Authentication is about validating  credentials like User Name  and password to verify user identity. The system determines whether user are what he says he is using his credentials.



                                         Authorization
Authorization, on the other hand, occurs after user identity has been successfully authenticated by the system, which ultimately gives user full permission to access the resources such as information, files, databases

                                      session

A session is a temporary and interactive information interchange between two or more communicating devices, or between a computer and user (see login session). A session is established at a certain point in time, and then ‘torn down’ - brought to an end - at some later point. 



                                                 cookie
Cookies are small files which are stored on a user's computer. They are designed to hold a modest amount of data specific to a particular client and website, and can be accessed either by the web server or the client computer.
